# Getting started

Leyline is a local interface for a configured pi coding-agent environment. It reads pi sessions, opens transcript detail, and uses pi runtime primitives to run prompts, tools, shell commands, forks, edits, compaction, and exports.

Fast path:

1. Install dependencies with `npm install`.
2. Start the browser app with `npm run dev`.
3. Open the Vite URL, usually `http://localhost:5173/`.
4. Choose a project cwd.
5. Create a pi session and submit a first prompt.
